## ⚡ Quick Start Deployment
1. **Clone the Repo:** `git clone https://github.com/YOUR_USERNAME/ai-millionaire-playbook.git`
2. **Initialize Supabase:** Run the SQL provided in `/infrastructure/schema.sql`.
3. **Deploy Edge Functions:** Use the provided GitHub Action in `.github/workflows/deploy.yml` to sync your local environment with Supabase.
